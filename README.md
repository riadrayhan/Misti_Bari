<h2>Mitha Bari</h2>

<P>Mitha Bari is an e-commerce based mobile application.In this app i use frontend and backend widget.</P>
<P>In backend i used </P>
<p>Firebase</p>
<p>Rest API</p>
<P>Shared Preference for Database</P>
<p>Json Data Parsing</p>
